# Java初级架构师入阶系列专栏

该系列只是对自己在java架构方面的现阶段理解，随着时间流逝，会持续校正。

写这个的目的不是给别人看，只是记录总结一下自己这方面的思考。

**为什么要按一个系列来写**？

这个不是随笔，而是系统性的思考整理而来。工作通常是解决零碎的小问题，很少能站在全局的角度考虑一些问题，通过一个系列，就是为了强迫自己在更多的角度，更全面的思考java初级架构师这个话题。
